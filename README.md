# Claude Code

## 新增 Plugin

1. 輸入 `/plugins`
2. 前往 `Marketplaces`
3. `+ Add Marketplace`
4. 輸入 `PeacePan/peace-agent-marketplace`
5. Space 選取要安裝的 Plugin
6. 輸入 `i` 進行安裝

## 更新 Plugin

1. 輸入 `/plugins`
2. 前往 `Marketplaces`
3. 選取 `wonderpet-agent-plugins`
4. 輸入 `u` 標記更新
5. 輸入 `Enter` 進行更新
6. 輸入 `/reload-plugins` 重新載入 Plugin
